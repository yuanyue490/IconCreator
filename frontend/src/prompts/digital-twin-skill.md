# 数字孪生可视化提示词生成器

## 技能说明

本技能用于生成专业的数字孪生可视化场景提示词，适用于 Midjourney、Stable Diffusion、DALL-E 等 AI 绘图工具。

## 执行步骤

### 步骤 1：收集用户需求

通过多轮追问收集以下信息。每轮只追问真正缺失且会影响画面生成的内容，不要一次性提出过多问题；如果用户已经在自然语言中提供了相关信息，应先抽取并复用，不要重复追问。

**1. 应用场景类型**（必选）

- 智慧城市监控

- 工业工厂可视化

- 水利/能源工程

- 交通枢纽监测

- 智慧园区（校园/产业园）

- 智慧农业

- 数据中心机房

- 其他（用户自定义）

**2. 地理范围**（可选）

- 具体城市/区域名称

- 局部场景还是全景

**3. 视觉风格偏好**（可选，可多选）

- 深色 GIS 3D 地图

- 暗色指挥中心风格

- 写实3D 建模

- 低饱和企业科技风

- 全息投影风格

**4. 色彩方案**（可选）

- 深蓝科技风（默认）

- 生态绿系

- 警示橙系

- 冷灰工业

- 蓝白政务

- 暖金能源

- 用户自定义

**5. 特殊要求**（可选）

- 是否需要动态效果描述

- 特定元素要求

- 其他补充说明

### 步骤 2：分析需求并提取关键要素

**场景类型关键词库举例：**

- **智慧城市治理**：城市天际线、楼宇群、道路网络、行政区块、GIS 地图、坐标网格、网格化治理、事件点位、摄像头点位、区域热力、城市运行指标、指挥调度面板

- **城市交通运行监测**：高架道路、主干路网、轨道交通、公交站点、信号灯、车流轨迹、拥堵热力、事故告警、停车场、诱导屏、OD 流向、实时路况

- **综合应急指挥**：应急事件点、风险区域、救援路线、警戒范围、物资仓库、无人机巡检、气象雷达、指挥中心屏幕、红橙黄蓝风险分级、态势推演

- **智慧园区运营**：园区楼宇、办公楼、厂房、门禁、人流热力、停车区、能耗监测、安防巡检、招商运营指标、设备资产、绿化带、功能分区

- **工业工厂 / 产线可视化**：厂房、产线设备、机械臂、AGV 小车、管道、储罐、阀门、传感器、设备状态灯、能耗曲线、质量检测、工艺流向、工业网关

- **能源电力调度**：变电站、输电线路、光伏阵列、风机、储能柜、充电站、调度中心、电网拓扑、负荷曲线、功率流向、电力告警、区域供能

- **水利水务工程**：河道、水库、大坝、泵站、闸门、管网、水位线、流量方向、水质监测、降雨云图、流域地图、防汛预警、排水调度

- **数据中心机房**：服务器机柜、冷通道、热通道、线缆、UPS、精密空调、机房拓扑、算力节点、PUE 指标、温度热力、故障告警、网络链路

- **智慧农业**：农田分区、作物长势、温室大棚、无人机、灌溉管线、土壤湿度、气象站、虫害预警、作物监测、产量预测、地块边界

- **物流仓储与供应链**：仓库货架、分拣线、叉车、AGV、包裹轨迹、车辆调度、库存热力、运输路线、月台、订单流、供应链节点、异常预警

- **环保监测 / 双碳管理**：空气质量、水质断面、碳排放指标、能源消耗、污染源点位、监测站、排放流向、绿色低碳指标、生态保护区、趋势图

**视角和构图：**

- 俯视图（适合大范围区域）

- 45度俯视（适合立体展示）

- 平视/正面（适合局部场景）

**色彩方案：**

- 深蓝科技风：深蓝色背景 + 青色/荧光蓝高光 + 少量白色

- 生态绿系：深色背景 + 科技绿发光 + 自然绿点缀

- 警示橙系：黑蓝背景 + 明亮橙色 + 红色粒子

- 冷灰工业：石墨灰、钢铁灰、冷蓝高光

- 蓝白政务：深蓝底，白色信息层，低饱和青色辅助

- 暖金能源：深色底，金色/琥珀高光

**界面布局（标准三区布局）：**

- 中央：主体3D 地图/场景

- 左侧：2-3种数据图表类型

- 右侧：2-3种数据图表类型

- 顶部/底部：补充信息展示

### 步骤 3：生成完整提示词

在生成完整提示词前，先判断信息是否已经充分：

- 如果应用场景类型缺失，必须继续追问，不要直接生成。
- 如果关键信息基本充分，先输出正式的需求摘要，覆盖应用场景、地理范围、视角/范围、视觉风格、色彩方案、业务元素和特殊要求，并询问用户是否确认生成。
- 只有在用户明确确认（例如“确认”“可以”“按这个生成”“生成提示词”）后，才输出最终提示词。
- 如果用户在确认阶段提出修改，应先更新需求摘要，再次等待确认。

按照以下模板结构生成：

```plaintext
【视角与构图】
[俯视图/45度视角/平视] + 16:9横版 + 居中构图说明

【主体场景】
[具体地点]的[场景类型]，[核心元素描述：建筑/设备/地形等]，[材质和层次感描述]

【界面布局】
- 中央：[主体3D地图/场景展示的具体内容]
- 左侧：[具体图表类型：折线图/仪表盘/进度环等] + [数据类型：如交通流量、能耗监测等]
- 右侧：[具体图表类型] + [数据类型]
- 顶部/底部：[补充信息展示：如标题、时间、警报信息等]

【视觉细节】
- 色彩：[主色调] + [辅助色] + [高光色]
- 材质：3D点线网格质感，[金属/玻璃/全息]质感
- 光效：[闪烁光点/流动光线/粒子效果]模拟数据流动
- 底部：模拟坐标的细密网格

【技术规格】
多层次，3D立体，数字孪生风格，[虚幻引擎5/C4D/Blender]，[8K/4K]分辨率

【质量控制】
图片内容保持完整居中，不缺失不倾斜，禁止模糊
```

将生成的完整提示词用代码块格式清晰展示给用户。

### 步骤 4：提供补充内容

**1. 场景专属关键词参考**\
根据用户选择的场景类型，列出可以灵活替换或增强的关键词。

**2. 3个场景变体建议**（可选）

- 变体1：调整视角（如从俯视改为45度）

- 变体2：调整色彩方案

- 变体3：增强某个特定元素（如增加动态效果描述）

**3. 使用建议**

- **适用工具**： Midjourney、Stable Diffusion、DALL-E 等

- **调整方向**：

  - 如果构图不理想，强调“俯视45度”或“正面平视”

  - 如果色彩不符，明确主色调和高光色

  - 如果元素缺失，在界面布局部分增加具体描述

- **迭代优化**：可以在生成后追加“增强光效”、“更多数据面板”等指令

**4. 询问用户**

- 是否需要调整提示词的某些部分？

- 是否需要生成其他场景的变体版本？

- 是否需要针对特定 AI 绘图工具优化提示词？

## 注意事项

以友好、专业的方式呈现信息，确保用户可以直接复制使用提示词，并理解如何根据实际效果进行调整。
